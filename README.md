# Vue Back + Front Playground

## Запуск приложения

Выполнить

```shell
pnpm install
pnpm dev
```

Открыть в браузере ссылку http://localhost:5173/

## Запуск тестов

Выполнить

```shell
pnpm run test
```

## Stack

### Server

NodeJS + Express + tRPC

### Client

Vite + Vue + Vue-query

### Common

jest, eslint, prettier
