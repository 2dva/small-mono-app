# Vue Back + Front Playground

## Запуск приложения

Выполнить

```shell
pnpm install
pnpm dev
```

Открыть в браузере ссылку http://localhost:8000/

## Запуск тестов

Выполнить

```shell
pnpm run test
```

## Stack

### Server

NodeJS + Express + tRPC + Prisma + SQLite

### Client

Vite + Vue + tanstack/vue-query + tRPC

### Common

jest, eslint, prettier
